## IronClaw 调研
- GitHub: https://github.com/nearai/ironclaw ⭐ 6.3k
- 定位：OpenClaw 的 Rust 重写版，主打隐私和安全，由 NEAR AI 开发
- 核心亮点：WASM 沙箱（比 Docker 更轻量的工具隔离）、凭据保护（密钥不暴露给工具+泄漏检测）、Prompt 注入防御（模式检测+内容消毒+策略引擎）、端点白名单、混合搜索（全文+向量+RRF）、自修复机制
- 技术栈：Rust + PostgreSQL + pgvector
- 与 OpenClaw 的差异：Rust vs TS、WASM vs Docker、PostgreSQL vs SQLite、单二进制分发
- 可借鉴点：WASM 工具沙箱设计、凭据泄漏检测、混合搜索架构、自修复机制
- 局限：需要 NEAR AI 账户，生态不成熟，目前主要参考价值是安全架构设计
